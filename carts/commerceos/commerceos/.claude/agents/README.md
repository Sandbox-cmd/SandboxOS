# raising a specialist — when a domain has earned one

raise an agent only when this project has accumulated a real domain
(canon + skills + repeated delegation). few beat many; the frontmatter is
the product, the persona prose is filler.

the template (one file, `<name>.md` beside this README):

    ---
    name: <lowercase-hyphen>
    description: <WHEN to delegate, with the words you'd actually say.
      "use when the task touches X, Y, or the user mentions Z." never
      summarize the workflow here — the body holds it.>
    tools: <the minimum. reviewers read, they don't write.>
    ---
    <the body is the agent's whole system prompt. short. rules:>
    - recall before deciding: sandbox recall QUERY --channel <channel>.
      canon before the internet.
    - your notebook is yours and uncitable — a notebook line worth
      trusting is a note worth staging: sandbox note "…".
    - never mark your own work verified; report receipts (exit codes,
      rendered output, paths) and let them speak.
