# rules — small root, lazy leaves

the root contract (AGENTS.md) stays under ~200 lines or it stops being
followed. anything narrower goes here as a topic file, scoped to the
paths where it bites:

    ---
    paths: ["src/payments/**"]
    ---
    <the rule, one screen at most.>

add a rule the SECOND time the same mistake happens, not before. delete
rules the agent follows without being told — a rule it never needed is
context spent on nothing.
